

## Installation

You can install the package via composer:

```bash
composer require iamowaisazam/starterpackage
```

## Usage (Just Like)

```php
$firstClass = new Owaisazam\StarterPackage\StarterPackage();
echo $firstClass->array_to_string(['1','2','3','4']);
```

## Testing

```bash
composer test
```
